# Agent Report Template

> 所有 agent 的标准回报格式。
> 这是 Codex subagent 经 `spawn_agent` 执行后的标准回报，不应省略测试与结果分析。

## Header

- `阶段/模块：`
- `agent：`
- `状态：PASS / FAIL / BLOCKED`

## 1. 已完成内容

- 改了什么
- 产出了什么
- 对应哪些文件

## 2. 测试

- `命令：`
- `结果：`
- `覆盖范围：`

## 3. 保持的 invariants

- 列出本次没有破坏的核心不变量

## 4. 风险 / TODO

- 当前剩余风险
- 未定义行为
- 需要后续确认的问题

## 5. 下一步

- 下一轮建议动作
- 是否可以交给下一个 agent

## 6. 禁止项

- 不允许只写“完成”
- 不允许隐藏失败
- 不允许把猜测写成事实
