## Cross-Thread Interface Notes

### Field / payload shape
- `cli_surface` 的 route handler 契约是字符串或 `Sequence[AlertView]`。
- `reactive_runtime` 的执行结果是结构化对象（`is_executed/callback_verified/abort_reason/...`）。
- 该形状差异尚未通过 adapter 固化，属于未解决的跨线程字段对接缺口。

### Enum
- `reactive_runtime` 已在本波次内验证 `ReactiveTriggerType` 与 `ReactiveCallbackType` 的映射。
- `cli_surface` 的 `AlertSeverity` 仅在本地渲染层验证，和其他线程枚举尚无已验证桥接关系（`not verified yet`）。

### TTL / timestamp unit
- `reactive_trigger.triggered_at` 为 timezone-aware `datetime`（UTC）。
- 本波次未验证其与其他模块中的 epoch seconds 字段（例如注册/执行链路常见的 Unix 秒）如何对齐。
- 因此 timestamp 单位对齐状态为 `not verified yet`。

### BPS / pct
- 本波次已验证的关系中没有落地 bps/pct 计算或跨线程传递。
- 是否存在 bps/pct 单位不一致，当前证据为 `not verified yet`。
