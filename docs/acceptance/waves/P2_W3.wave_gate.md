# P2_W3 Wave Gate

## Wave Goal

Prove fork/testnet E2E with real RPC and Uniswap V2-compatible route.

## Scope

```text
real RPC / fork RPC
-> reserve and price read
-> PreRegistrationCheck
-> ExecutionPlan
-> register
-> entry
-> exit
-> event sync
-> ExecutionRecord recovery
```

## Allowed Paths

- RPC provider and runtime paths listed in prompts
- event syncer paths listed in prompts
- fork/testnet configuration examples without secrets
- `docs/acceptance/threads/phase2_wave3/**`

## Forbidden Paths

- committing private keys, RPC secrets, or `.env` values
- cross-chain execution
- Uniswap V3 implementation
- Aave implementation
- production mainnet execution without explicit approval

## Required Contracts

- `docs/contracts/phase2_core_execution_loop.contract.md`
- `docs/contracts/pre_registration_check.contract.md`
- `docs/contracts/execution_compiler.contract.md`
- `docs/contracts/execution_layer.contract.md`
- `docs/contracts/phase2_event_syncer.contract.md`

## Required Smoke Tests

```powershell
python scripts/workflow.py check pre_registration_check --execute --strict
python scripts/workflow.py check execution_layer --execute --strict
python scripts/workflow.py check reactive_runtime --execute --strict
```

Implementation-specific smoke should prove one fork/testnet loop from register through event-synced `Closed`.

## Exit Criteria

- [ ] real RPC or fork RPC reads block, balance, allowance, gas, reserve, and price inputs.
- [ ] PreRegistrationCheck uses RPC truth.
- [ ] register tx or fork receipt is captured.
- [ ] entry and exit receipts are captured.
- [ ] EventSyncer reconstructs `ExecutionRecord`.
- [ ] rerunning sync is idempotent.
- [ ] evidence includes chain id, tx hashes or fork receipts, and final state.

## Rollback / No-Go Conditions

- execution truth comes from an indexer instead of RPC and chain events.
- event sync produces duplicate records.
- fork/testnet evidence is missing but E2E is marked complete.

## Handoff to Next Wave

W4 may harden Reactive adapter and export closure using W3's fork/testnet evidence as the baseline.
