# 统一验收命令建议

下面不是强制命令，而是建议你在真实仓库里统一约定的命令名。

## Python 层
```bash
pytest tests/unit
pytest tests/integration
ruff check .
mypy backend
```

## 合约层
```bash
forge test
forge build
```

## CLI / smoke
```bash
python -m backend.cli --help
agent-cli decision dry-run --strategy <id>
agent-cli approval show <intent-id>
agent-cli export json <intent-id>
```

## 验收记录建议
每次把命令、commit、结果、截图路径写回：
- thread_test_evidence.md
- wave_integration_test_matrix.md
- wave_exit_report.md
