# 建议执行顺序

## 每个线程结束时
1. 填 `01_thread_acceptance/01_thread_acceptance_checklist.md`
2. 填 `01_thread_acceptance/02_thread_delivery_note.md`
3. 填 `03_thread_handoff/01_thread_to_thread_handoff.md`
4. 若接口变化，填 `03_thread_handoff/03_interface_change_request.md`

## 每个 Wave 结束时
1. 填 `02_wave_acceptance/01_wave_gate_checklist.md`
2. 填 `02_wave_acceptance/02_wave_integration_test_matrix.md`
3. 填 `02_wave_acceptance/03_wave_risk_register.md`
4. 填 `04_wave_handoff/01_wave_to_wave_handoff.md`
5. 填 `04_wave_handoff/03_wave_exit_report.md`

## 推荐 owner
- 线程内验收：该线程负责人
- Wave gate：你 / tech lead / integration owner
- 接口冻结记录：上游模块 owner
- Wave-to-Wave handoff：当前 Wave gate owner
